# 🐾 Sistema de Petshop em Python

Este projeto consiste em um sistema simples de gerenciamento de petshop desenvolvido em Python, com foco na aplicação de conceitos de **Programação Orientada a Objetos (POO)**.

## 📌 Objetivo

Praticar e consolidar conceitos fundamentais de POO, como:

* Encapsulamento
* Abstração
* Organização em classes
* Separação de responsabilidades

---

## ⚙️ Funcionalidades

O sistema permite:

### 👤 Clientes

* Cadastro de clientes
* Listagem de clientes

### 🐶 Pets

* Cadastro de pets vinculados a um cliente
* Listagem de pets com identificação do dono

### 🛠️ Serviços

* Listagem de serviços disponíveis:

  * Banho
  * Tosa
  * Banho + Tosa

### 📅 Agendamentos

* Agendar serviços para pets
* Listar agendamentos
* Marcar agendamentos como concluídos

### 💰 Faturamento

* Cálculo do faturamento total com base nos serviços concluídos

---

## 🧱 Estrutura do Projeto

O sistema é composto pelas seguintes classes:

* `Cliente` → Representa um cliente
* `Pet` → Representa um pet vinculado a um cliente
* `Servico` → Representa um serviço oferecido
* `Agendamento` → Controla os agendamentos
* `Petshop` → Classe principal que gerencia todo o sistema

---

## 🛠️ Tecnologias Utilizadas

* Python 3
* `dataclasses` (para simplificar a criação de classes)
